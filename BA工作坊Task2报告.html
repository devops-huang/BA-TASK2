三、基础技术指标理论与实现

3.1 RSI (相对强弱指标)

术语解释：

  • RSI (Relative Strength Index)：由Welles Wilder于1978年提出，用于衡量股价变动的速度和幅度，判断超买超卖状态。
  • 超买区：RSI > 70，表示股价可能过高，有回调风险。
  • 超卖区：RSI < 30，表示股价可能过低，有反弹机会。

计算方法：

1. 计算价格变动：Δprice = close[t] - close[t-1]
2. 计算平均涨幅(Average Gain)：AG = (前13天的总涨幅 + 当日涨幅) / 14
3. 计算平均跌幅(Average Loss)：AL = (前13天的总跌幅 + 当日跌幅) / 14
4. 计算相对强度(RS)：RS = AG / AL
5. 计算RSI：RSI = 100 - (100 / (1 + RS))

作用：

  • 判断超买超卖状态
  • 识别背离信号（价格创新高/新低，但RSI未创新高/新低）
  • 辅助判断趋势反转

Python实现代码：

def calculate_rsi(df, period=14):
    """
    计算RSI指标
    
    参数:
    - df: DataFrame，包含'close'列
    - period: RSI计算周期，默认14天
    
    返回:
    - DataFrame，新增'RSI'列
    """
    # 计算价格变动
    delta = df['close'].diff()
    
    # 分离涨幅和跌幅
    gain = (delta.where(delta > 0, 0)).rolling(window=period).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(window=period).mean()
    
    # 计算RS和RSI
    rs = gain / loss
    df['RSI'] = 100 - (100 / (1 + rs))
    
    return df

3.2 MACD (指数平滑异同移动平均线)

术语解释：

  • MACD (Moving Average Convergence Divergence)：由Gerald Appel于1970年代提出，是一种趋势跟踪动量指标。
  • DIF线：快线，通常是EMA(12) - EMA(26)。
  • DEA线：慢线，通常是DIF的EMA(9)。
  • MACD柱：DIF - DEA，用柱状图表示。

计算方法：

1. 计算快线EMA(12)：EMA_fast = close * (2/(12+1)) + EMA_prev * (1 - 2/(12+1))
2. 计算慢线EMA(26)：EMA_slow = close * (2/(26+1)) + EMA_prev * (1 - 2/(26+1))
3. 计算DIF：DIF = EMA_fast - EMA_slow
4. 计算DEA：DEA = DIF的EMA(9)
5. 计算MACD柱：MACD = (DIF - DEA) * 2

作用：

  • 识别趋势方向（DIF在DEA上方为多头，下方为空头）
  • 捕捉买卖信号（DIF上穿DEA为金叉，买入信号；DIF下穿DEA为死叉，卖出信号）
  • 判断动能强弱（MACD柱的绝对值越大，动能越强）

Python实现代码：

def calculate_macd(df, fast=12, slow=26, signal=9):
    """
    计算MACD指标
    
    参数:
    - df: DataFrame，包含'close'列
    - fast: 快线周期，默认12
    - slow: 慢线周期，默认26
    - signal: 信号线周期，默认9
    
    返回:
    - DataFrame，新增'MACD_DIF'、'MACD_DEA'、'MACD'列
    """
    # 计算EMA
    ema_fast = df['close'].ewm(span=fast, adjust=False).mean()
    ema_slow = df['close'].ewm(span=slow, adjust=False).mean()
    
    # 计算DIF和DEA
    df['MACD_DIF'] = ema_fast - ema_slow
    df['MACD_DEA'] = df['MACD_DIF'].ewm(span=signal, adjust=False).mean()
    
    # 计算MACD柱
    df['MACD'] = (df['MACD_DIF'] - df['MACD_DEA']) * 2
    
    return df

3.3 布林带 (Bollinger Bands)

术语解释：

  • 布林带 (Bollinger Bands)：由John Bollinger于1980年代提出，由三条轨道线组成，用于衡量价格波动范围和趋势强弱。
  • 上轨 (Upper Band)：中轨 + 2倍标准差，通常表示压力位。
  • 中轨 (Middle Band)：通常是20日简单移动平均线(SMA)。
  • 下轨 (Lower Band)：中轨 - 2倍标准差，通常表示支撑位。

计算方法：

1. 计算中轨：MB = SMA(close, 20)
2. 计算标准差：std = std(close, 20)
3. 计算上轨：UB = MB + (2 * std)
4. 计算下轨：LB = MB - (2 * std)

作用：

  • 判断价格波动范围（价格通常在上轨和下轨之间波动）
  • 识别突破信号（价格突破上轨可能继续上涨，跌破下轨可能继续下跌）
  • 判断趋势强弱（布林带收窄表示盘整，扩张表示趋势走强）

Python实现代码：

def calculate_bollinger_bands(df, period=20, std_dev=2):
    """
    计算布林带指标
    
    参数:
    - df: DataFrame，包含'close'列
    - period: 中轨周期，默认20
    - std_dev: 标准差倍数，默认2
    
    返回:
    - DataFrame，新增'BB_upper'、'BB_middle'、'BB_lower'列
    """
    # 计算中轨
    df['BB_middle'] = df['close'].rolling(window=period).mean()
    
    # 计算标准差
    std = df['close'].rolling(window=period).std()
    
    # 计算上轨和下轨
    df['BB_upper'] = df['BB_middle'] + (std * std_dev)
    df['BB_lower'] = df['BB_middle'] - (std * std_dev)
    
    return df

四、扩展指标：KDJ随机指标

4.1 KDJ指标介绍

术语解释：

  • KDJ指标：由George Lane于1950年代提出，是一种动量振荡指标，用于判断超买超卖状态和趋势反转。
  • K值：快速线，反映当前收盘价在最近9天价格波动范围中的位置。
  • D值：慢速线，是K值的3日指数移动平均。
  • J值：J = 3K - 2D，反映K和D的偏离程度。

计算方法：

1. 计算RSV：RSV = (close - LLV(low, 9)) / (HHV(high, 9) - LLV(low, 9)) * 100
   其中：LLV = 最近9天的最低值，HHV = 最近9天的最高值
   
2. 计算K值：K = (2/3) * K_prev + (1/3) * RSV
3. 计算D值：D = (2/3) * D_prev + (1/3) * K
4. 计算J值：J = 3K - 2D

作用：

  • 判断超买超卖状态（K/D > 80为超买，K/D < 20为超卖）
  • 捕捉买卖信号（K上穿D为金叉，买入信号；K下穿D为死叉，卖出信号）
  • 识别背离信号（价格创新高/新低，但KDJ未创新高/新低）

Python实现代码：

def calculate_kdj(df, period=9):
    """
    计算KDJ指标
    
    参数:
    - df: DataFrame，包含'high'、'low'、'close'列
    - period: 计算周期，默认9
    
    返回:
    - DataFrame，新增'KDJ_K'、'KDJ_D'、'KDJ_J'列
    """
    # 计算最低价和最高价
    low_list = df['low'].rolling(window=period).min()
    high_list = df['high'].rolling(window=period).max()
    
    # 计算RSV
    rsv = (df['close'] - low_list) / (high_list - low_list) * 100
    
    # 计算K值和D值
    df['KDJ_K'] = rsv.ewm(com=2, adjust=False).mean()
    df['KDJ_D'] = df['KDJ_K'].ewm(com=2, adjust=False).mean()
    
    # 计算J值
    df['KDJ_J'] = 3 * df['KDJ_K'] - 2 * df['KDJ_D']
    
    return df
