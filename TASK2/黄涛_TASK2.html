二、基础技术指标理论

2.1 RSI (相对强弱指标)

定义: RSI (Relative Strength Index) 是由Welles Wilder提出的动量振荡指标，用于衡量股价变动的速度和幅度。

计算方法:

RS = 平均涨幅 / 平均跌幅
RSI = 100 - (100 / (1 + RS))

其中：
- 平均涨幅 = 过去N天上涨幅度的平均值
- 平均跌幅 = 过去N天下跌幅度的平均值
- N通常为14

Python实现代码:

def calculate_rsi(data, period=14): """计算RSI指标""" delta = data.diff() gain = (delta.where(delta > 0, 0)).rolling(window=period).mean() loss = (-delta.where(delta < 0, 0)).rolling(window=period).mean() rs = gain / loss rsi = 100 - (100 / (1 + rs)) return rsi df['RSI'] = calculate_rsi(df['close'])

作用:

  • 超买超卖判断: RSI > 70 表示超买，股价可能回调；RSI < 30 表示超卖，股价可能反弹。
  • 背离信号: 股价创新高但RSI未创新高，为顶背离，看跌；股价创新低但RSI未创新低，为底背离，看涨。
  • 中线突破: RSI突破50中轴线，表明趋势可能反转。
实际应用: RSI是一种领先指标，能在股价反转前给出信号。但需注意，在强趋势市场中，RSI可能长时间停留在超买或超卖区，因此应结合其他指标综合判断。RSI适合震荡市使用，在趋势明确的市场中效果较差。

2.2 MACD (移动平均收敛/发散指标)

定义: MACD (Moving Average Convergence Divergence) 是由Gerald Appel提出的趋势跟踪动量指标，基于移动平均线的交叉原理。

计算方法:

DIF = EMA(close, 12) - EMA(close, 26)
DEA = EMA(DIF, 9)
MACD = (DIF - DEA) × 2

其中：
- EMA: 指数移动平均线
- 12, 26, 9 为常用参数

Python实现代码:

def calculate_macd(data, fast=12, slow=26, signal=9): """计算MACD指标""" ema_fast = data.ewm(span=fast, adjust=False).mean() ema_slow = data.ewm(span=slow, adjust=False).mean() dif = ema_fast - ema_slow dea = dif.ewm(span=signal, adjust=False).mean() macd = (dif - dea) * 2 return dif, dea, macd df['MACD_DIF'], df['MACD_DEA'], df['MACD'] = calculate_macd(df['close'])

作用:

  • 趋势判断: DIF在DEA上方为多头市场，下方为空头市场。
  • 买卖信号: DIF上穿DEA为金叉（买入信号），下穿为死叉（卖出信号）。
  • 背离信号: 股价与MACD柱背离，预示趋势反转。
  • 零轴突破: DIF突破零轴，表明中长期趋势反转。
实际应用: MACD是一种滞后指标，适合趋势明显的市场。在震荡市中，MACD会频繁发出虚假信号，因此应结合RSI等指标，或在趋势明确时使用。MACD的金叉和死叉信号在趋势明确时非常有效，但在震荡市中容易产生"骗线"。

2.3 布林带 (Bollinger Bands)

定义: 布林带是由John Bollinger提出的波动率指标，基于移动平均线和标准差原理。

计算方法:

中轨 = MA(close, N)
上轨 = 中轨 + (K × std(close, N))
下轨 = 中轨 - (K × std(close, N))

其中：
- N通常为20
- K通常为2
- std: 标准差

Python实现代码:

def calculate_bollinger(data, period=20, std_dev=2): """计算布林带指标""" middle = data.rolling(window=period).mean() std = data.rolling(window=period).std() upper = middle + (std * std_dev) lower = middle - (std * std_dev) return upper, middle, lower df['BB_upper'], df['BB_middle'], df['BB_lower'] = calculate_bollinger(df['close'])

作用:

  • 超买超卖判断: 股价触及上轨为超买，触及下轨为超卖。
  • 波动率判断: 布林带收窄表示波动率下降，即将变盘；布林带开口表示波动率上升，趋势明确。
  • 趋势判断: 股价在中轨上方为多头市场，下方为空头市场。
  • 突破信号: 股价突破上轨或下轨，表明趋势加速。
实际应用: 布林带适合震荡市和趋势市。在震荡市中，可采用"高抛低吸"策略（下轨买入，上轨卖出）；在趋势市中，可结合突破信号交易。但需注意，突破上轨不一定是买入信号，可能是强势上涨的开始。布林带的"收口"和"开口"是重要的变盘信号。

四、扩展指标：KDJ随机指标

4.1 KDJ指标介绍

定义: KDJ指标（随机指标）是由George Lane提出的动量振荡指标，基于收盘价在最近N天最高价和最低价区间内的相对位置。

计算方法:

RSV = (CLOSE - LLV(LOW, N)) / (HHV(HIGH, N) - LLV(LOW, N)) × 100
K = EMA(RSV, M1)
D = EMA(K, M2)
J = 3 × K - 2 × D

其中：
- N通常为9
- M1, M2通常为3
- LLV: 最近N天最低值
- HHV: 最近N天最高值

Python实现代码:

def calculate_kdj(high, low, close, period=9, m1=3, m2=3): """计算KDJ指标""" low_list = low.rolling(window=period).min() high_list = high.rolling(window=period).max() rsv = (close - low_list) / (high_list - low_list) * 100 k = rsv.ewm(com=m1-1, adjust=False).mean() d = k.ewm(com=m2-1, adjust=False).mean() j = 3 * k - 2 * d return k, d, j df['KDJ_K'], df['KDJ_D'], df['KDJ_J'] = calculate_kdj(df['high'], df['low'], df['close'])

作用:

  • 超买超卖判断: K、D > 80 为超买，K、D < 20 为超卖。
  • 买卖信号: K线上穿D线为金叉（买入信号），下穿为死叉（卖出信号）。
  • 背离信号: 股价与J值背离，预示趋势反转。
  • 极端值: J > 100 为超买，J < 0 为超卖，比K、D更敏感。
实际应用: KDJ是一种敏感的振荡指标，适合短线交易。但需注意，在强趋势市场中，K、D会长时间停留在超买或超卖区，因此应结合MACD等趋势指标综合判断。J值对价格变化更敏感，可提前给出信号，但也更容易产生虚假信号。KDJ指标在震荡市中表现较好，在趋势市中容易产生"钝化"现象。
