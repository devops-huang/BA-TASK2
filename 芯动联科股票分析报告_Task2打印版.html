2.1 RSI (相对强弱指标)

📖 术语解释：RSI (Relative Strength Index)

提出者: Welles Wilder (1978年)
用途: 衡量股价变动的速度和幅度，判断超买或超卖状态

计算公式:
RSI = 100 - [100 / (1 + RS)]
RS = 平均涨幅 / 平均跌幅

其中:
- 平均涨幅 = 过去N天（通常14天）上涨幅度的平均值
- 平均跌幅 = 过去N天下跌幅度的绝对值的平均值

参数设置:
- 常用周期：14天（默认）、6天（短期）、24天（长期）

判断标准:
- RSI > 70: 超买区域，股价可能回调或下跌
- RSI < 30: 超卖区域，股价可能反弹或上涨
- RSI = 50: 多空分界线

作用:
1. 判断超买超卖状态
2. 识别趋势反转信号（顶背离、底背离）
3. 辅助判断买卖时机

局限性:
- 在强趋势市场中，RSI可能长时间停留在超买或超卖区域
- 对短期波动敏感，可能产生虚假信号

2.2 MACD (指数平滑异同移动平均线)

📖 术语解释：MACD (Moving Average Convergence Divergence)

提出者: Gerald Appel (1970年代)
用途: 捕捉趋势反转和动量变化

计算公式:
DIF = EMA(12) - EMA(26) (快线)
DEA = EMA(DIF, 9) (慢线)
MACD = (DIF - DEA) × 2 (柱状图)

参数设置:
- 快线周期：12
- 慢线周期：26
- 信号线周期：9
- 这是MACD的默认参数设置（12/26/9）

判断标准:
- DIF > DEA（金叉）: 买入信号，多头趋势
- DIF < DEA（死叉）: 卖出信号，空头趋势
- MACD > 0: 多头市场
- MACD < 0: 空头市场

作用:
1. 判断趋势方向（多头/空头）
2. 识别趋势反转信号（金叉/死叉）
3. 衡量趋势强度（MACD柱状图的高度）
4. 识别顶背离和底背离

局限性:
- 在震荡市中，MACD可能频繁发出虚假信号
- 对短期波动不敏感，存在滞后性

2.3 布林带 (Bollinger Bands)

📖 术语解释：布林带 (Bollinger Bands)

提出者: John Bollinger (1980年代)
用途: 衡量价格波动率和超买超卖状态

计算公式:
中轨（MB）= MA(close, N) (N日收盘价的简单移动平均)
上轨（UB）= MB + (K × σ) (中轨 + K倍标准差)
下轨（LB）= MB - (K × σ) (中轨 - K倍标准差)

其中:
- N: 移动平均周期，默认20天
- K: 标准差倍数，默认2倍
- σ: N日收盘价的标准差

参数设置:
- 中轨周期：20天（默认）
- 标准差倍数：2倍（默认），可调整为1.5倍或2.5倍

判断标准:
- 股价触及上轨: 超买状态，可能回调
- 股价触及下轨: 超卖状态，可能反弹
- 布林带收窄: 波动率降低，预示即将变盘
- 布林带放大: 波动率增加，趋势明确

作用:
1. 判断超买超卖状态
2. 识别趋势突破信号（突破上轨或下轨）
3. 衡量波动率变化（布林带宽度）
4. 辅助判断买卖时机

局限性:
- 在强趋势市场中，股价可能长时间沿着上轨或下轨运行
- 对突变行情反应滞后

2.4 KDJ (随机指标) - 扩展指标 (Task2要求)

📖 术语解释：KDJ (随机指标)

提出者: George Lane (1950年代)
用途: 判断超买超卖状态和趋势反转信号

计算公式:
RSV = (CLOSE - LLV(LOW, N)) / (HHV(HIGH, N) - LLV(LOW, N)) × 100

K值 = 2/3 × 前一日K值 + 1/3 × 当日RSV
D值 = 2/3 × 前一日D值 + 1/3 × 当日K值
J值 = 3 × K值 - 2 × D值

其中:
- N: 周期，默认9天
- LLV(LOW, N): N日内最低价的最小值
- HHV(HIGH, N): N日内最高价的最大值

参数设置:
- 默认周期：9天
- K值平滑参数：2/3（默认）
- D值平滑参数：2/3（默认）

判断标准:
- K > D（金叉）: 买入信号，多头趋势
- K < D（死叉）: 卖出信号，空头趋势
- J > 100: 超买区域，股价可能回调
- J < 0: 超卖区域，股价可能反弹

作用:
1. 判断超买超卖状态
2. 识别趋势反转信号（金叉/死叉）
3. 捕捉短期买卖信号
4. 识别顶背离和底背离

局限性:
- 在震荡市中，KDJ可能频繁发出虚假信号
- 对长期趋势判断能力较弱

4.1 RSI计算代码

def calculate_rsi(df, period=14):
    """
    计算RSI指标
    
    参数:
    - df: DataFrame，包含'close'列
    - period: RSI计算周期，默认14天
    
    返回:
    - df: 添加'RSI'列的DataFrame
    """
    # 计算价格变化
    delta = df['close'].diff()
    
    # 分离涨幅和跌幅
    gain = delta.where(delta > 0, 0)  # 涨幅（正数）
    loss = -delta.where(delta < 0, 0)  # 跌幅（转为正数）
    
    # 计算平均涨幅和平均跌幅（使用指数移动平均）
    avg_gain = gain.ewm(com=period-1, adjust=False).mean()
    avg_loss = loss.ewm(com=period-1, adjust=False).mean()
    
    # 计算RS和RSI
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    
    df['RSI'] = rsi
    return df

4.2 MACD计算代码

def calculate_macd(df, fast=12, slow=26, signal=9):
    """
    计算MACD指标
    
    参数:
    - df: DataFrame，包含'close'列
    - fast: 快线周期，默认12
    - slow: 慢线周期，默认26
    - signal: 信号线周期，默认9
    
    返回:
    - df: 添加'MACD_DIF', 'MACD_DEA', 'MACD'列的DataFrame
    """
    # 计算EMA
    ema_fast = df['close'].ewm(span=fast, adjust=False).mean()
    ema_slow = df['close'].ewm(span=slow, adjust=False).mean()
    
    # 计算DIF（快线）
    df['MACD_DIF'] = ema_fast - ema_slow
    
    # 计算DEA（慢线）
    df['MACD_DEA'] = df['MACD_DIF'].ewm(span=signal, adjust=False).mean()
    
    # 计算MACD柱状图
    df['MACD'] = (df['MACD_DIF'] - df['MACD_DEA']) * 2
    
    return df

4.3 布林带计算代码

def calculate_bollinger_bands(df, period=20, std_dev=2):
    """
    计算布林带指标
    
    参数:
    - df: DataFrame，包含'close'列
    - period: 移动平均周期，默认20天
    - std_dev: 标准差倍数，默认2倍
    
    返回:
    - df: 添加'BB_upper', 'BB_middle', 'BB_lower'列的DataFrame
    """
    # 计算中轨（MA）
    df['BB_middle'] = df['close'].rolling(window=period).mean()
    
    # 计算标准差
    std = df['close'].rolling(window=period).std()
    
    # 计算上轨和下轨
    df['BB_upper'] = df['BB_middle'] + (std * std_dev)
    df['BB_lower'] = df['BB_middle'] - (std * std_dev)
    
    # 计算布林带宽度（波动率指标）
    df['BB_width'] = (df['BB_upper'] - df['BB_lower']) / df['BB_middle']
    
    return df

4.4 KDJ计算代码

def calculate_kdj(df, period=9):
    """
    计算KDJ指标
    
    参数:
    - df: DataFrame，包含'high', 'low', 'close'列
    - period: 周期，默认9天
    
    返回:
    - df: 添加'KDJ_K', 'KDJ_D', 'KDJ_J'列的DataFrame
    """
    # 计算RSV
    low_list = df['low'].rolling(window=period).min()
    high_list = df['high'].rolling(window=period).max()
    
    rsv = (df['close'] - low_list) / (high_list - low_list) * 100
    
    # 计算K值和D值（使用指数移动平均）
    df['KDJ_K'] = rsv.ewm(com=2, adjust=False).mean()
    df['KDJ_D'] = df['KDJ_K'].ewm(com=2, adjust=False).mean()
    
    # 计算J值
    df['KDJ_J'] = 3 * df['KDJ_K'] - 2 * df['KDJ_D']
    
    return df
